Conjure by macourtney

Conjure is a Rails like framework written entirely in Clojure. The goal is to have a framework which is close enough to Rails that anyone familiar with Rails will be able to pick up Conjure very quickly.

Dependencies

All dependencies for Conjure will be included with the release. You will only need to be sure you have a recent version of java (1.5 or later). Included dependencies: * Clojure * Clojure Contrib * H2 Database Engine * Jetty

Install

There are no releases for Conjure yet. The only way to get it is to check out the code from gethub. Once you have checked out the code from gethub, simply run ant from the root Conjure directory. Ant will build a jar file called Conjure and place it in your target directory. To create a new web app, move the conjure.jar file into the directory where you want the web app to reside, and run 'java -jar conjure.jar '. Conjure will create a new directory with all of the files you need to start your new web app.
